# Clipboard Landing Page

## 📋 Descripción
Sitio web desarrollado con HTML y CSS para el desafío **Clipboard Landing Page** de Frontend Mentor. Este proyecto demuestra el uso de etiquetas HTML y propiedades CSS para crear una landing page moderna y responsiva para una aplicación de portapapeles.

## 🚀 Características
- **Diseño responsivo** - Se adapta a diferentes tamaños de pantalla (375px - 1440px)
- **Estados hover** - Efectos interactivos en botones y enlaces
- **Diseño moderno** - Utiliza CSS Grid, Flexbox y variables CSS
- **Optimizado** - Código limpio y bien estructurado
- **Accesible** - Cumple con estándares de accesibilidad web

## 🎯 Desafío de Frontend Mentor
Este proyecto es una solución al desafío [Clipboard Landing Page](https://www.frontendmentor.io/challenges/clipboard-landing-page-5cc9bccd6c4c91111378ecb9) de Frontend Mentor.

### Requisitos del Desafío:
- ✅ Construir la landing page según el diseño proporcionado
- ✅ Hacer el diseño responsivo para móvil y desktop
- ✅ Implementar estados hover para elementos interactivos
- ✅ Usar solo HTML y CSS

## 🚀 Cómo Ejecutar
1. Abre el archivo `index.html` en tu navegador
2. O usa Live Server en VS Code/Cursor para desarrollo
